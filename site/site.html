Preventing paste into an input field

This example demonstrates how to stop a user from pasting content into a HTML input form field using JavaScript. Check the HTML and JavaScript panes for more details or scroll to the bottom to find a code example.

This example was created by Claire Parker-Jones and I only forked it in case it was deleted or moved.

Code Example

<input type="text" id="paste-no-event" />
<script>
  const pasteBox = document.getElementById("paste-no-event");
  pasteBox.onpaste = e => {
    e.preventDefault();
    return false;
  };
</script>

Bypassing in Code

Any of the following lines of code can remove the protection.


document.getElementById("paste-no").onpaste={};
document.getElementById("paste-no").onpaste=null;
document.getElementById("paste-no").onpaste=new function(){};

I can bypass the protection from the console or by DOM editing


document.getElementById("paste-no-event").value="hello";
